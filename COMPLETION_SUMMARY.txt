TROUBLESHOOTING
===============

Q: "ModuleNotFoundError: No module named 'requests'"
A: Run: pip install -r requirements.txt

Q: "sqlite3.OperationalError: database is locked"
A: Run: rm sarscope/sarscope.db (resets database)

Q: "Website returns 403/429 (Blocked)"
A: Increase MAX_DELAY in config.py

Q: "Price not extracted"
A: Check website structure, use custom CSS selector

Q: "Import errors on startup"
A: Activate virtual environment: source venv/bin/activate
